# Reboot Recovery Results

This file is generated by `scripts/dws-reboot-drill.sh verify`.

No automated reboot drill has been recorded in this commit yet.

Run:

```bash
./scripts/dws-reboot-drill.sh snapshot
# reboot the host
./scripts/dws-reboot-drill.sh verify
```

The script stores the pre-reboot snapshot under
`${XDG_STATE_HOME:-$HOME/.local/state}/dev-workspace/reboot-drill/` so it
survives the reboot, then overwrites this file with the latest markdown report.
